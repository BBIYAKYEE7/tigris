# TIGRIS 주점 운영 가이드

고려대학교 아마추어 아이스하키 동아리 **TIGRIS** 주점의 온라인 주문 시스템 운영 설명서입니다.  

> 💬 **개발자가 한 번 설치해 준 후**, 여러분이 주점을 운영할 때 사용하는 가이드입니다.

---

## 🌐 접속 방법

개발자가 미리 설치해 뒀으므로, 주점 운영할 때는 아래 주소로 접속하면 됩니다:

### 손님 화면 (메뉴판)
```
https://love-tigris.vercel.app/
```
- 손님들이 스마트폰으로 접속
- QR코드 제작 후 테이블 마다 부착
- 테이블 번호 입력 후 메뉴 선택

### 관리자 화면 (주문 관리)
```
https://love-tigris.vercel.app/admin
```
- 카운터에서 노트북/태블릿으로 접속
- 주문 목록 확인 및 결제 처리

---

## 📖 주점 운영 방법

### 👥 손님이 사용하는 방법 (손님 화면)

1. **테이블 번호 입력** - 앉은 테이블 번호 입력 (예: 5번)
2. **메뉴 선택** - 메뉴판에서 수량 선택
3. **주문하기** - "주문하기" 버튼 클릭
4. **결제 대기** - 결제 대기 목록에서 주문 확인
5. **결제** - 카운터에서 직원이 처리할 때까지 대기

> 💡 같은 테이블 같은 번호를 입력한 손님이 보는 주문 목록은 같습니다 (합석 가능)

### 💼 직원이 사용하는 방법 (관리자 화면)

1. **주문 목록 보기** - 관리자 화면 열기
2. **테이블별 주문 확인** - 어느 테이블이 뭘 주문했는지 확인
3. **결제 처리** - "결제완료" 버튼 누르기
4. **소리 알림** - 새 주문이 들어오면 자동으로 알림음 재생 (🔊)

---

## 🎯 주요 기능

### 손님 화면의 특징
- 🏓 **테이블 공유 기능**: 같은 번호 입력한 손님들이 주문 목록을 공유
- 🔄 **자동 새로고침**: 4초마다 자동으로 주문 목록 갱신
- 📱 **반응형 디자인**: 휴대폰, 태블릿, 컴퓨터 모두 지원
- 🎨 **핑크 테마**: 귀여운 핑크 디자인

### 관리자 화면의 특징
- 📊 **전체 주문 목록**: 모든 테이블의 주문을 한눈에 보기
- 🔔 **음성 알림**: 새 주문이 들어오면 알림 소리
- ⚡ **빠른 결제 처리**: 버튼 하나로 주문 "결제완료" 처리
- 📈 **수익 현황**: 오늘의 매출 자동 계산

## 🔧 운영 중 문제 해결

### 손님 화면이 안 열려요
💻 **해결 방법:**
1. 인터넷 연결 확인
2. 다시 새로고침 (F5 또는 브라우저 새로고침 버튼)
3. 주소가 맞는지 확인 (개발자한테 물어보세요)

### 주문이 안 보여요
💻 **해결 방법:**
1. 손님들이 정말 주문했는지 확인
2. 관리자 화면을 새로고침 (F5)
3. 개발자에게 알려주세요

### 관리자 화면에서 알림음이 안 들려요
💻 **해결 방법:**
1. 노트북/태블릿 음량 확인 (음소거 풀기)
2. 브라우저 음량 확인 (음소거 아이콘 클릭)
3. 여전히 안 들리면 개발자에게 연락하세요

### 손님들이 다른 주문을 봐요
💻 **상황:**
- 같은 테이블 번호를 입력한 손님들은 **같은 주문 목록**을 봐야 합니다
- 혹시 손님이 다른 번호를 입력했나 확인하세요

### 더 이상 주문이 안 들어와요
💻 **해결 방법:**
1. 손님들이 정말 주문하고 있는지 확인
2. 손님 화면의 URL이 맞는지 확인
3. 개발자에게 연락하세요

## 📝 메뉴 또는 테이블비 수정하고 싶어요

메뉴 이름, 가격, 테이블비를 수정할 때:

📋 **수정할 파일:** `frontend/src/lib/menu.ts`

🔧 **수정 내용:**
- 메뉴 이름 변경
- 메뉴 가격 변경
- 테이블비 변경 (`TABLE_FEE_AMOUNT`: 현재 5,000원)

> 💡 수정이 복잡하면 **개발자에게 물어보세요**.
> 개발자가 수정한 후 배포하면 자동으로 반영됩니다.

- **테이블비**: 5,000원으로 설정됨
